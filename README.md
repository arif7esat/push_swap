## ⚙️ Algorithmic Architecture & First Principles
This project focuses on advanced data sorting using two stacks (Stack A and Stack B) with a severely restricted set of allowed operations. Instead of relying on conventional sorting methods, a completely custom algorithm was engineered from **first principles** using mathematical logic to achieve maximum efficiency under strict instruction limits.

### Core Engineering Features
* **Custom Mathematical Modeling:** Designed proprietary sorting logic tailored for large, dynamic datasets, proving competence in solving complex algorithmic bottlenecks.
* **Aggressive Big O Optimization:** Iteratively refined the time complexity to meet aggressive performance benchmarks, a critical skill for developing resource-efficient logic in **constrained embedded systems and microcontrollers**.
* **Memory-Efficient Data Structures:** Architected stack-based data structures in pure C to process dynamic inputs with minimal computational overhead.

## 🛡️ Robustness & System Stability
